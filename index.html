HTML

  • Start with the !DOCTYPE, html, head, and body elements to define the basic structure of your webpage.
  • Use html, head, and body tags to structure your document.
  • Set the title tag inside the head tag to give your webpage a title.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS can change text colors, backgrounds, fonts, and more
  • CSS selects HTML elements to apply styles
  • CSS controls element positioning and spacing using properties

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript has data types like strings, numbers, and booleans
  • Use if statements to make decisions in your code
  • Functions are reusable blocks of code
